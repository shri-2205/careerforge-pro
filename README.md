# ResumeAI Pro — Vite + MERN Stack

## Setup

### Backend
```bash
cd backend
cp .env.example .env
# Edit .env with your values
npm install
node server.js
```

### Frontend
```bash
cd frontend
npm install
npm run dev
# Opens at http://localhost:3000
```

## .env File
```
PORT=5000
MONGODB_URI=mongodb+srv://user:pass@cluster.mongodb.net/resumeai_pro
JWT_SECRET=resumeai_super_secret_jwt_key_here_make_it_long
GROQ_API_KEY=gsk_your_groq_key_here
FRONTEND_URL=http://localhost:3000
```

## Tech Stack
- Frontend: React 18 + Vite + Bootstrap 5
- Backend: Node.js + Express
- Database: MongoDB + Mongoose
- AI: Groq API (Llama 3.3 70B)
- Auth: JWT
